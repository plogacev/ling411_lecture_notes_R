Chapter 3 ## Determining Data Types

When unsure, we can ask R about the type of a variable:

> my_apples <- 1
> my_oranges <- "1.1"
> typeof(my_apples)
## [1] "double"
> typeof(my_oranges)
## [1] "character"
> typeof(as.double(my_oranges))
## [1] "double"
