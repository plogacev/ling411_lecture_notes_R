1.2 Installing and Setting up RStudio

Currently, the most convenient interface to R is the RStudio GUI. You can either install it locally, or use it online on RSudio Cloud. RStudio Cloud.

1.2.1 Local RStudio installation

  • Typically, RStudio is run on a local computer.

Please download R and RStudio separately from: - https://cran.r-project.org/ - https://rstudio.com/products/rstudio/download/ (download ‘RStudio Desktop’)

1.2.2 RStudio Cloud

You can use the RStudio cloud service for free after getting an account to https://rstudio.cloud.
